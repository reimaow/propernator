Modrinth

  1. Download Modrinth app
  2. Sign in with your account if needed
  3. Click on the + symbol on the left side to create an instance
  4. Name it whatever, but it has to be the Fabric loader and version 1.21.11
  5. Once created, click on the instance and on the right side click Install Content
  6. Find these three mods: Fabric API, Placeholder #1, Placeholder #2
  7. Wait for the install to finish and you are done

You can now launch Minecraft from Modrinth with the Play button and join the server.

Curseforge

  1. Download Curseforge app
  2. In the app, go to the Minecraft section and click on the Create to make a new profile
  3. Name it whatever, but it has to be the Fabric loader and version 1.21.11
  4. Once created, click on the profile and on the right side click Add Content
  5. Find these two mods: Placeholder #1, Placeholder #2
  6. Wait for the install to finish and you are done

You can now launch Minecraft from Curseforge with the Play button and join the server.
